# Testing-with-Playwright
Playwright UI testing

## Usage Instructions
### Installation
```
git clone https://github.com/Jorazon/Testing-with-Playwright.git
cd Testing-with-Playwright
npm i
npx playwright install
```

### Running tests
```
npx playwright test
```

### Showing test report
```
npx playwright show-report
```
